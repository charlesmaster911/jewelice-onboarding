✅ AMPM 글로벌 — 2026.04.23 1차 미팅 결과 NEW

📋 미팅 총평: 전체 퍼널 구조·리타겟팅·소재 개선·보고 체계·데이터 분석 방향 모두 체계적 설명. 당사 광고 문제점(CTR 하락·장바구니 이탈·구매 전환 저하) 진단 방향도 대체로 정확. 다만 실제 운영 역량·제안 신뢰도는 1차 제안안(4/24)으로 별도 검증 필요.
구분 내용
① 현재 광고 진단 • CTR 하락 → 소재 피로도·크리에이티브 개선 필요 • 장바구니 매출 > 구매완료 매출 차이 큼 → 구매 전환 단계 이탈 심각 • 광고 지속 집행으로 ROAS는 유지되나, 신규 유입·전환 구조 재정비 필요
② 제안 운영 방향 3단계 퍼널 구조 ① 신규 유입 — 논타겟/관심사 기반 브랜드·제품 인지 확보 ② 장바구니·관심고객 리타겟팅 — 이탈자·결제 직전 이탈자 구매 전환 유도 ③ 재구매·확장 구매 — 기존 고객 연관/세트 상품 추가 구매 유도 병행 운영: 애드부스트 · 쇼핑 프로모션 · 전환 집중 배너 · 카탈로그 성과형 세팅
③ 소재·메시지 전략 • 퍼널별 맞춤 메시지 — 브랜딩 / 제품 강점 / 할인 프로모션 / 후기·만족도 • CTR 하락 원인 중 하나로 소재 부족·동일 소재 반복 지목 • 이미지뿐 아니라 실사용 콘텐츠·인플루언서 영상·레시피 콘텐츠 적극 활용 방향
④ 자사 상황 공유 • 기존 광고 체계 없이 운영·예산 편차 큼 → 정형화된 운영·보고 체계 필요 • 내부 광고 전담 인력 부족 → 외부 대행사 디테일한 관리 필요 • 쥬얼아이스 대표님 성향상 주간 단위 보고 중시 · 일별 데이터는 내부 확인 • 주요 SKU 3종: 듀오메이커 · 아블러 · 아블러 제로 → SKU별 소구 포인트 재정리 필요
⑤ 상품·타겟 확장 • 듀오메이커 — 위스키 얼음 → 커피·우유·막걸리·팥빙수·아이스크림 확장형 아이스 메이커 콘텐츠 가능 • 아블러 제로 — 40-50대 여성층 반응 양호 (세척 편의성·직관성 장점) • 여성 타겟 확장 가능성 → 향후 콘텐츠·광고 방향에 반영
⑥ 추가 논의 • 유튜브 쇼핑 · 인플루언서 · 공동구매 · 라이브커머스 연계 확장 검토 • 경쟁사 분석 · 시장 내 포지셔닝 분석 요청 • 강민경·빠니보틀·전현무 브랜드 노출 이력 존재 → 미활용 자산 우회 활용 방안 검토 필요
⑦ 후속 액션 ✔ 금일(4/23) 광고 계정 권한 AMPM 전달 → 현재 데이터 정밀 분석 요청 ✔ 4/24(금) 1차 제안안 수령 예정 (예산안·매체 운영안·초기 집행 방향) ✔ 내부: 광고 소재·콘텐츠 자산 정리 후 순차 전달 ✔ 이후 AMPM 진행 결정 여부 판단 예정

✅ 이프로애드 (EPROAD) — 2026.04.24 1차 미팅 결과 NEW

📋 미팅 총평: 10년+ 업력 중간급 대행사. 사무실 상주 30~40명 · 재대행 원칙 금지 · OOH·제작 강점 / 온라인 PPL·브랜디드 콘텐츠는 타 업체와 유사 수준 솔직히 인정. 쥬얼아이스 대표님 3대 조건(레퍼런스 증빙 / 과장급 고정 담당 / 재대행 금지) 중 2개 충족. 레퍼런스북은 최근 2년 2건 선별 전달 예정.
구분 내용
① 업체 프로필 • 이프로애드 (EPROAD) · 서울 강남구 선릉로 642 3층 • 참석자: 팀장급 + 재호 주임 (메인 PM 예정) • 업력 10년+ · 내부 30~40명 상주 • 조직: 광고사업팀 / 컨텐츠미디어(마케팅·디자인) / 캠페인본부(기획1·2팀) / 프로덕션본부(제작·연출) / 스튜디오본부(편집·AI모션) • 주요 레퍼런스: 유안타증권(1년), 지자체·공공기관 5억 프로젝트 多 (커머스·소비재는 약할 가능성)
② 쥬얼아이스 현황 공유 (팀장 → EPROAD) • 대표님 AI 기반 자체 광고 운영 중 (메타 집중 · 네이버 약함) — "대충 하면 바로 캐치함" • 객단가 높음 이유: 의료등급 실리콘 + 특허 보유 • 브랜딩 부족 — 제품은 알려졌으나 "쥬얼아이스" 회사 네임 약함 • 강민경·빠니보틀·전현무 자연 노출 존재 (미활용 자산) • 10월 해외 마케팅(인디고고 미국 → 베트남·대만) · 기후테크 사업 연계 • 선물용 세트(20만원대) 패키지 준비 중
③ 제안 운영 방향 우선순위 1 — 온라인 퍼포먼스 • 네이버 + 메타 + 구글 + 카카오비즈보드 기본 패키지 • 기존 영상 재활용 소재 제작 먼저, 신규 제작은 순차 우선순위 2 — OOH (여름 6~9월 이벤트성 집중) • 강남권 아파트 엘리베이터 광고 • 영화관 광고(전국) · 편의점 광고 • 식당·술집 키오스크 광고 • 서울역·용산역 조명(이미지) 광고 • 팀장급 직접 OOH 총괄 (10년+ 전문성) 우선순위 3 — 브랜딩 IMC • 제품 홍보 ≠ 브랜딩 영상 별도 필요 • 반복 노출 + 다채널 동시 집행으로 "쥬얼아이스 = 투명 얼음 원조" 각인
④ 인플루언서·PPL 전략 • 수수료 베이스 → 대부분 인플루언서 기피 (확실한 섭외비 선호 · 플러스 알파 포기 어려움) • 매니아층 타겟(위스키·홈바 계열)만 추천 — 어쿠스틱 링크 사례처럼 1회 1억 매출 가능 • 연예인 PPL은 상하반기 각 1명급 이벤트성 • 강민경·빠니보틀·전현무 자연 노출 → 라이선스 구매로 우회 활용 가능 (브랜디드 컨텐츠 비용의 30~50%) • 지자체 홍보영상처럼 협의 가능한 인플루언서만 선별 섭외
⑤ 예산 운용 • 고정 월 3,000만원 (팀장 제시) • 이벤트 건별 포함 최대 5,000만원 • 탄력 운용 가능 — 월 1,000 + 쉬기, 특정 달 집중 등 • 계약 형태: 월 단위, 최소 3개월 유지 권장 (성과 가시화 기간) • 네이버 캐시충전 / 메타 15% 수수료 등은 계약 체결 후 협의
⑥ 이프로애드 자기 포지셔닝 "CJ·메가급 메이저는 아니다. 꼼꼼함·집중·누락방지로 승부하는 중간급." • 팀장 멘트: "재호 주임과 수년째 일했는데 한 번도 사고 난 적 없음. 누가 놓치면 누가 챙김" • 약점 자인: 온라인 PPL·브랜디드 콘텐츠는 타 업체와 유사 수준 • 강점: OOH 10년+ · 재대행 없는 내부 자원 · 일·주·월간 보고 체계
⑦ 후속 액션 ✔ 이프로애드 → 최근 2년 레퍼런스 2건 선별 전달 (커머스·소비재 우선 요청 필요) ✔ EPROAD → 2차 미팅용 제안서 준비 — 쥬얼아이스 궁극 목표 기반 기획안 ✔ 팀장 → 월요일(4/28) 전후 대표님 보고 ✔ 팀장 → 3개 업체(AMPM·MAZE·EPROAD) 비교표 완성 ✔ 2차 미팅 (대표님 + 각 팀장 포함) 일정 조율 예정 ✔ 2차 미팅 시 PM 이름·직급·투입률(%) 확정 요청
