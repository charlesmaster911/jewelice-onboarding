① AMPM글로벌 광고대행사 미팅 요약 🥉

2026-04-23 (목) · 박지현 단독 참석
미팅 총평 — 전체 퍼널 구조·리타겟·소재 개선·보고 체계·데이터 분석 방향은 비교적 체계적으로 설명. CTR 하락·장바구니 이탈·구매 전환 저하 진단 방향 적정. 다만 실제 운영 역량과 제안 신뢰도는 별도 검증 필수.

1. 현재 광고 진단

  • 1~4월 데이터 기준 CTR 하락 → 소재 피로도 개선 필요
  • 장바구니 매출 > 구매완료 매출 → 결제 단계 이탈 큼
  • ROAS 유지되나 신규 유입·전환 구조 재정비 필요

2. 제안 운영 방향 (3단 퍼널)

  • ① 신규 유입 (논타겟/관심사) — 인지도 확보
  • ② 장바구니/관심고객 리타겟팅 — 구매 전환 유도
  • ③ 재구매·확장 — 연관·세트 상품 추가 구매
  • + 애드부스트·쇼핑 프로모션·전환 배너·카탈로그 성과형 병행

3. 소재·메시지 전략

  • 퍼널별 맞춤 메시지(브랜딩·강점·할인·후기) 운영 필요
  • CTR 하락 원인 — 소재 부족·반복 사용 가능성
  • 실사용 콘텐츠·인플루언서 영상·레시피 콘텐츠 적극 활용

4. 자사 상황 공유

  • 광고 체계·예산 편차 → 정형화 필요
  • 광고 전담 인력 부족 → 외부 디테일 관리 필요
  • 대표님 주간 단위 보고 중시 전달
  • 듀오메이커·아블러·아블러 제로 SKU별 소구점 재정리 필요

5. 상품·타겟 확장 아이디어

  • 듀오메이커 — 위스키 외 커피·우유·막걸리·팥빙수·아이스크림 확장
  • 아블러 제로 — 40~50대 여성 반응 양호 · 세척 편의성·직관성 강점
  • 여성 타겟 확장 가능성 충분

6. 추가 논의

  • 유튜브 쇼핑·인플루언서·공동구매·라이브커머스 연계 검토
  • 경쟁사·시장 포지셔닝 분석 요청
  • 강민경·빠니보틀·전현무 자산 우회 활용 방안 검토

② 이프로애드(EPROAD) 광고대행사 미팅 요약 🥇

미팅 총평 — 대표님 3대 조건을 3사 중 유일하게 모두 명확 충족. 25페이지 상세 제안서 + 자사몰 95% 의존 리스크·GA4 미설치·브랜드 키워드 과다 지출 즉석 진단. 네이버 플레이스 5개 광고 선제 추천. 박지현 자발 평: "팀장님이 신뢰 간다. 꼼꼼하게 잘해줄 것 같다."

1. 회사 개요

  • 2005년 설립 · 20년차 · 대표 김병현
  • 직원 45명(2026.01) · 5개 본부

2. 강점·레퍼런스 20+

  • 아모레·카멜·롯데카드·케이뱅크·이케아·VISA·엘빈즈·스메그·필립스·KOTRA·KB생명·아가방 외
  • 카멜 SA 6개월 ROAS 400%↑
  • 엘빈즈 가입 CPA → LTV 전환 성공
  • VISA 글로벌 옥외·팝업 통합

3. 미팅 중 즉석 진단 (자발 적극성)

  • 자사몰 95% 의존 = 리스크 → 채널 다변화 필요
  • GA4 미설치 → 통합 지표 관리 즉시 셋업 필요
  • 브랜드 키워드 과다 지출 → 캠페인 분리 재세팅
  • 네이버 플레이스 5개 광고(강남 키오스크) 선제 추천
  • 강남권·40~50대 여성 타겟 즉석 이해

4. 비용 조건

  • 네이버 수수료 0% · 구글·메타 각 15%
  • 이미지 무상 + 영상 2회 무상 수정 · 쇼츠 건별
  • GA4·GTM·에이스카운터 무상
  • 주간+월간+비정기 리포트 (코멘트·이슈·액션 첨부)

5. 투입 10명 (과장급 6명 고정)

  • 본부장 1(10년) + 팀장 3(10·11·4년)
  • 과장 2(12·2년) + 대리 3(4·5·5년) + 주임 1(3년)
  • 김** 과장(디자인 12년) + 오** 팀장(영상 퀄리티)

6. 콘텐츠 제작 역량

  • 콘텐츠제작본부 자체 (디자인+영상 제작팀)

  • 소재 컨셉 재설정 — CTR 0.9% → 2.1% 개선
